# Student Data Manager (Java)

A simple Java console application that collects and displays student information using object-oriented programming concepts.

## Project Description

The Student Data Manager allows users to enter the total number of students in a class, input each studentâ€™s name, roll number, and marks, store the data using an ArrayList, and display all student records in a clean format.
